# Cozy Loop 🧶

A soft, pastel catalog website for **Cozy Loop** — a handmade crochet shop by Joseph Paul (@cozyloop_2212).

## What's inside
- Crochet treats, animal keychains (Cozy Paws), and accessories
- Product catalog with MYR (RM) prices
- Orders handled via Instagram DM

## Stack
Static site — plain HTML, CSS, and JavaScript. No build step, no dependencies.

```
index.html    # all sections (hero, about, catalog, contact)
styles.css    # cozy pastel theme
script.js     # renders the product catalog
```

## Run locally
Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

## Hosting
Deployed via **GitHub Pages** from the `main` branch.
